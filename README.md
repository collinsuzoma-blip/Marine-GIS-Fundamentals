# Marine GIS Fundamentals – Hydrographic Mapping

## Overview
This project demonstrates foundational Geographic Information System (GIS) techniques applied to marine and hydrographic datasets. The project integrates bathymetric, hydrographic, and topographic datasets to produce a professional cartographic product for St. John’s Harbour, Newfoundland.

The workflow illustrates how marine geospatial datasets with different coordinate systems and formats can be integrated within ArcGIS to support hydrographic analysis.

## Study Area
St. John’s Harbour and surrounding coastal environment, Newfoundland and Labrador, Canada.

## Data Sources
- Bathymetry (BAG / GeoTIFF)
- Nautical charts
- Shipwreck locations
- Grab sample stations
- Sound velocity profile (SVP) points
- Coastal infrastructure datasets

## Tools & Software
- ArcGIS Pro
- ArcGIS Desktop
- Microsoft Visio
- Microsoft Excel

## Workflow
1. Import spatial datasets into ArcGIS Pro
2. Verify coordinate reference systems
3. Apply projection-on-the-fly
4. Create layer files for consistent symbology
5. Visualize bathymetric raster surfaces
6. Overlay hydrographic point and vector datasets
7. Produce final cartographic map

## Results
The project produced a hydrographic map displaying:
- Bathymetric depth variation
- Shipwreck locations
- Grab sampling stations
- Sound velocity profile points
- Survey boundary
